## 项目介绍
### 淘兴趣 淘兴趣平台为用户提供了一个兴趣讨论的平台。本系统要实现的业务是： 1. 用户可以进入该系统看到各种各样的趣点。 2. 用户可以通过点击查看趣点的详情信息和其他用户发布的微博。 3. 如果用户已登录，可以评论微博，如果是未登录状态，则提示登录。 4. 用户通过订阅趣点来发布自己的微博。 技术选型：Struts2 + Spring + Hibernate 特点：B/S架构，技术偏复杂，工作量较大。
### 轻松短租网 轻松短租网是一个房屋租住平台，该平台为房客出行租房提供了便利。本系统要实现的业务是： 1. 用户注册登录后可以创建房屋，待管理员审核后正式发布出去，供其它用户预订。 2. 用户可以根据地点和名称来搜索房屋，查看房屋的详细介绍。 3. 用户可以下订单预订房屋，当房东受理订单后，表示预订成功。 技术选型：Struts2 + JDBC 特点：B/S架构，业务较复杂，工作量适中。
### 跳蚤市场 本项目为用户提供了进行二手商品交易所需要的所有功能。本系统要实现的业务是： 1. 用户可以搜索商品，查看商品详情。 2. 用户登陆后，可以进行发布商品，修改商品，修改密码、联系方式。 3. 用户可以查看自己发布的商品、订单和消费记录，管理自己的订单和留言。 4. 管理员可以管理商品分类、管理用户、审核商品、修改配置等。 技术选型：Android + Http通信 + JSP + Servlet + JDBC 特点：C/S + B/S架构，典型的电商网站+手机APP，工作量较大。
### 计步器 计步器项目主要用来记录用户的运动参数。本系统要实现的业务是： 1. 用户可以开启计步器，记录自己的运动参数，查看自己的历史运动记录，运动轨迹等。 2. 登陆的用户，可以添加好友，可以获取与好友之间的运动距离排名。 3. 登陆的用户，可以进入个人中心，查看所获取的荣誉以及各项运动数据统计的结果。 技术选型：Android + Http通信 + JSP + Servlet + JDBC 特点：C/S + B/S架构，重点在于Android传感器应用，手机端较重，工作量适中。
